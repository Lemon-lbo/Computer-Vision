# Computer Vision – Assignment 2

This repository contains implementations of core computer vision concepts, including spatial filtering, feature extraction, and neural network analysis.

* Applied and compared spatial filters (Box, Gaussian, Laplacian, Sobel)
* Explored the Viola–Jones algorithm, including integral images and Haar-like features
* Computed HOG features with gradient-based orientation binning
* Analyzed a CNN architecture, including layer dimensions and parameter counts

The work emphasizes both theoretical understanding and practical implementation of key vision techniques.
